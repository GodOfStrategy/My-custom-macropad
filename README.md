# My-custom-macropad
Made with and for Blueprint @ Hackclub. It is my personal hackpad, with the keys W,A,S,D with various utility.

My hackpad uses:
1. XIAO-SAMD21-DIP as the MCU (sourced from Hackclub hq)
2. 4 push switches as my keys.(sourced from Hackclub hq)
3. 4 Cherry MX keycaps. (3D printed or sourced from Hackclub HQ)
